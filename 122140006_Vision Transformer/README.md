Proyek ini bertujuan untuk melakukan eksperimen komparatif antara dua arsitektur Vision Transformer, yaitu **ViT** dan **Swin Transformer**, menggunakan dataset **CIFAR-10**. Eksperimen meliputi proses:
- Fine-tuning model pre-trained
- Evaluasi performa (accuracy, precision, recall, F1-score)
- Perhitungan waktu inferensi
- Visualisasi learning curve dan confusion matrix
- Perbandingan hasil antar model

Langkah-Langkah Menjalankan Program:
1. Clone Repository (jika dari GitHub)
```bash
git clone <repo-anda>
cd <repo-anda>

2. Install Depedencies
Install semua depedensi menggunakan: 
pip install -r requirements.txt

3. Pastikan perangkat mendukung GPU (jika tidak, otomatis menggunakan CPU). Menggunakan:
